# Database Columns Required for Coordinator Dashboard Page

## Overview
This document outlines all database tables and columns needed to support the Coordinator Dashboard functionality shown in the interface.

## Required Tables and Columns

### 1. STUDENTS Table
```sql
Students (
    StudentID           INT PRIMARY KEY IDENTITY,
    StudentName         NVARCHAR(255) NOT NULL,
    StudentNumber       NVARCHAR(50) UNIQUE NOT NULL,
    UniversityID        INT FOREIGN KEY,
    ProgramID           INT FOREIGN KEY,
    YearOfStudy         INT CHECK (YearOfStudy >= 1 AND YearOfStudy <= 6),
    GPA                 DECIMAL(3,2) CHECK (GPA >= 0 AND GPA <= 4.0),
    RiskLevel           NVARCHAR(20) CHECK (RiskLevel IN ('Safe', 'At Risk', 'Critical')),
    ContactEmail        NVARCHAR(255),
    ContactPhone        NVARCHAR(20),
    EmergencyContact    NVARCHAR(255),
    EmergencyPhone      NVARCHAR(20),
    DateEnrolled        DATETIME NOT NULL,
    LastLoginDate       DATETIME,
    LastActivityDate    DATETIME,           -- For "LAST ACTIVITY" column
    IsActive            BIT DEFAULT 1,
    CreatedAt           DATETIME DEFAULT GETDATE(),
    UpdatedAt           DATETIME DEFAULT GETDATE()
)
```

### 2. UNIVERSITIES Table
```sql
Universities (
    UniversityID        INT PRIMARY KEY IDENTITY,
    UniversityName      NVARCHAR(255) NOT NULL UNIQUE,
    UniversityCode      NVARCHAR(10),           -- e.g., UCT, UJ, UKZN, Wits
    IsActive            BIT DEFAULT 1,
    CreatedAt           DATETIME DEFAULT GETDATE()
)
```

### 3. PROGRAMS Table  
```sql
Programs (
    ProgramID           INT PRIMARY KEY IDENTITY,
    ProgramName         NVARCHAR(255) NOT NULL,
    ProgramCode         NVARCHAR(20),           -- e.g., BSC-CS, BCOM, LLB
    DurationYears       INT DEFAULT 3,
    IsActive            BIT DEFAULT 1,
    CreatedAt           DATETIME DEFAULT GETDATE()
)
```

### 4. SUPPORT_REQUESTS Table (for Active Interventions)
```sql
SupportRequests (
    RequestID           INT PRIMARY KEY IDENTITY,
    StudentID           INT FOREIGN KEY REFERENCES Students(StudentID),
    RequestType         NVARCHAR(50),           -- Academic, Wellness, Financial, etc.
    Priority            NVARCHAR(20),           -- Low, Medium, High, Critical
    Status              NVARCHAR(50),           -- Open, In Progress, Resolved, Closed
    Description         NVARCHAR(MAX),
    AssignedPartnerID   INT FOREIGN KEY,
    CreatedAt           DATETIME DEFAULT GETDATE(),
    ResolvedAt          DATETIME,
    IsActive            BIT DEFAULT 1
)
```

### 5. ACTIVITY_LOGS Table (for Last Activity tracking)
```sql
ActivityLogs (
    ActivityID          INT PRIMARY KEY IDENTITY,
    StudentID           INT FOREIGN KEY REFERENCES Students(StudentID),
    ActivityType        NVARCHAR(100),          -- Login, Assessment, Support Request, etc.
    ActivityDescription NVARCHAR(500),
    ActivityDate        DATETIME DEFAULT GETDATE(),
    IPAddress           NVARCHAR(45),
    UserAgent          NVARCHAR(500)
)
```

## Dashboard Statistics - Required Calculations

### Top Statistics Bar:
```sql
-- TOTAL STUDENTS (130)
SELECT COUNT(*) FROM Students WHERE IsActive = 1

-- AT-RISK STUDENTS (39)  
SELECT COUNT(*) FROM Students 
WHERE IsActive = 1 AND RiskLevel IN ('At Risk', 'Critical')

-- AVERAGE GPA (54.8%)
SELECT AVG(GPA) FROM Students WHERE IsActive = 1

-- ACTIVE INTERVENTIONS (6)
SELECT COUNT(*) FROM SupportRequests 
WHERE Status IN ('Open', 'In Progress') AND IsActive = 1
```

### Risk Distribution Chart:
```sql
SELECT 
    RiskLevel,
    COUNT(*) as Count,
    CAST(COUNT(*) * 100.0 / SUM(COUNT(*)) OVER() AS DECIMAL(5,2)) as Percentage
FROM Students 
WHERE IsActive = 1 
GROUP BY RiskLevel
```

### Students Overview Table:
```sql
SELECT 
    s.StudentID,
    s.StudentName,                          -- STUDENT column
    u.UniversityName,                       -- UNIVERSITY column  
    u.UniversityCode,                       -- For abbreviations (UCT, UJ, etc.)
    p.ProgramName,                          -- PROGRAM column
    CASE 
        WHEN s.YearOfStudy = 1 THEN '1st Year'
        WHEN s.YearOfStudy = 2 THEN '2nd Year'  
        WHEN s.YearOfStudy = 3 THEN '3rd Year'
        WHEN s.YearOfStudy = 4 THEN '4th Year'
        WHEN s.YearOfStudy = 5 THEN '5th Year'
        WHEN s.YearOfStudy = 6 THEN 'Final Year'
    END as Year,                            -- YEAR column
    s.GPA,                                  -- GPA column
    s.RiskLevel,                            -- RISK LEVEL column
    CASE 
        WHEN DATEDIFF(MINUTE, s.LastActivityDate, GETDATE()) < 60 
            THEN CAST(DATEDIFF(MINUTE, s.LastActivityDate, GETDATE()) AS VARCHAR) + ' mins ago'
        WHEN DATEDIFF(HOUR, s.LastActivityDate, GETDATE()) < 24 
            THEN CAST(DATEDIFF(HOUR, s.LastActivityDate, GETDATE()) AS VARCHAR) + ' hour' + 
                 CASE WHEN DATEDIFF(HOUR, s.LastActivityDate, GETDATE()) > 1 THEN 's' ELSE '' END + ' ago'
        ELSE CAST(DATEDIFF(DAY, s.LastActivityDate, GETDATE()) AS VARCHAR) + ' day' +
             CASE WHEN DATEDIFF(DAY, s.LastActivityDate, GETDATE()) > 1 THEN 's' ELSE '' END + ' ago'
    END as LastActivity                     -- LAST ACTIVITY column
FROM Students s
    LEFT JOIN Universities u ON s.UniversityID = u.UniversityID
    LEFT JOIN Programs p ON s.ProgramID = p.ProgramID
WHERE s.IsActive = 1
ORDER BY s.LastActivityDate DESC
```

## Filter Dropdowns Data:

### Universities Dropdown:
```sql
SELECT UniversityID, UniversityName 
FROM Universities 
WHERE IsActive = 1 
ORDER BY UniversityName
```

### Programs Dropdown:
```sql
SELECT ProgramID, ProgramName 
FROM Programs 
WHERE IsActive = 1 
ORDER BY ProgramName
```

### Risk Levels Dropdown:
```sql
-- Static values: 'Safe', 'At Risk', 'Critical'
SELECT DISTINCT RiskLevel 
FROM Students 
WHERE IsActive = 1 AND RiskLevel IS NOT NULL
```

## Search Functionality:

### Keyword Search (Student name, Student ID):
```sql
SELECT * FROM Students s
    LEFT JOIN Universities u ON s.UniversityID = u.UniversityID
    LEFT JOIN Programs p ON s.ProgramID = p.ProgramID
WHERE s.IsActive = 1 
    AND (s.StudentName LIKE '%{search_term}%' 
         OR s.StudentNumber LIKE '%{search_term}%'
         OR CAST(s.StudentID AS VARCHAR) LIKE '%{search_term}%')
```

## Additional Useful Columns for Dashboard Enhancement:

### Students Table - Additional tracking:
- ProfilePictureURL      NVARCHAR(500)    -- For student avatars  
- AttendanceRate         DECIMAL(5,2)     -- Percentage attendance
- NextMeetingDate        DATETIME         -- Upcoming appointments
- NotificationCount      INT DEFAULT 0    -- Unread notifications

### Dashboard Preferences Table:
```sql
CoordinatorDashboardSettings (
    SettingID           INT PRIMARY KEY IDENTITY,
    CoordinatorID       INT,
    DefaultUniversityFilter INT,
    DefaultProgramFilter    INT,
    DefaultRiskFilter       NVARCHAR(20),
    ItemsPerPage           INT DEFAULT 10,
    AutoRefreshEnabled     BIT DEFAULT 1,
    CreatedAt              DATETIME DEFAULT GETDATE()
)
```

## Indexes for Performance:

```sql
-- For fast filtering and searching
CREATE INDEX IX_Students_RiskLevel ON Students(RiskLevel)
CREATE INDEX IX_Students_UniversityID ON Students(UniversityID)  
CREATE INDEX IX_Students_ProgramID ON Students(ProgramID)
CREATE INDEX IX_Students_LastActivityDate ON Students(LastActivityDate DESC)
CREATE INDEX IX_Students_Name_Number ON Students(StudentName, StudentNumber)

-- For statistics calculations
CREATE INDEX IX_Students_Active_Risk ON Students(IsActive, RiskLevel)
CREATE INDEX IX_SupportRequests_Status ON SupportRequests(Status, IsActive)
```
